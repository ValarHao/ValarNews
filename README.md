# ValarNews
一款基于[知乎日报API](https://github.com/izzyleung/ZhihuDailyPurify/wiki/%E7%9F%A5%E4%B9%8E%E6%97%A5%E6%8A%A5-API-%E5%88%86%E6%9E%90)开发的阅读类APP  
框架采用 MVP + RxJava + Retrofit + OkHttp + Glide  
声明: 本项目仅做学习交流使用，欢迎大家star、fork，API数据内容所有权归原作者所有，请勿用于其他用途
